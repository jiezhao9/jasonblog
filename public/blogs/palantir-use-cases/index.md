# 01 预警相关
## 金融类案例
- 摩根士丹利（Morgan Stanley）：利用Palantir Metropolis平台监控全球交易活动，建立复杂的风险预警工作流，实时检测市场异常、潜在违规交易和操作风险，并向合规团队自动发送警报。

- 渣打银行：使用Palantir构建反洗钱预警系统，实时分析交易网络，自动识别可疑模式并生成警报案例，大幅提升调查人员处理高风险交易的效率。

- Credit Suisse (瑞士信贷 )：未授权交易监控 - 针对内部合规风险，系统监控交易员的行为模式（如非正常工作时间交易、超出限额的仓位），一旦发现偏离正常模式的潜在违规操作，立即向合规官发送警报。

## 制造与零售类案例
- 空中客车（Airbus）：通过实时监控飞行传感器数据，系统能在零部件真正失效前检测到异常振动或温度变化，向航空公司发出预警，建议在飞机落地后立即检查特定部件，避免航班因突发故障而取消；根据历史数据（天气、罢工）识别推测交付延迟风险。
- 喜力啤酒：识别供应链中断风险；识别库存不平衡问题（某些仓库的产品太多或者太少）。
- Hertz 赫兹租车 ：持续监测天气、航班，客户需求， 识别导致运营中断的风险 。
- BP石油：提前预测某个钻头可能失效，在真的坏掉之前，提前进行维修。
